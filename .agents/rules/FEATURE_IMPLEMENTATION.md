# Feature Implementation Workflow

Before coding:

1. Read architecture
2. Read domain
3. Check state

Then:

1. Analyze existing implementation
2. Identify affected files
3. Create implementation plan
4. Implement
5. Verify consistency

Never jump directly into coding.
